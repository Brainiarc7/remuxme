# remuxme
Ffmpeg process manager


## installation
```
$ npm install https://github.com/remux-io/remuxme.git -g
```
or
```
$ git clone https://github.com/remux-io/remuxme.git
$ cd remuxme
$ npm install
```

## start cli

```
$ remux
```
non global
```
$ [remuxmeHome]/bin/remux
```
